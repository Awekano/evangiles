# Évangiles
Ceci est un petit projet codé en C qui donne un verset des 4 Évangiles au hasard
Pour le moment seul les 6 premiers chapitre de l'Évangile de **Matthieu** y sont présent
A l'avenir tous les Évangiles y seront présent
## Installation 
Installer à partir du .zip puis éxécuter **script.sh** qui fera tout, ensuite dans votre shell tapez : "livre_evangile" afin de démarrer
